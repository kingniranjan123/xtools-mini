🎥 FFmpeg Setup

FFmpeg is strictly required for video conversion, watermarking, audio extraction, and portrait splitting.

📖 Manual Setup Guide
1. Download GPL Release from BtbN FFmpeg.
2. Extract to `C:\ffmpeg`.
3. Open Windows Start → "Environment Variables" → System Properties.
4. Edit "Path" array and append: `C:\ffmpeg\bin`.
5. Restart your terminal & the application.
🔴 YT-DLP Updater

YouTube constantly changes its decipher layout. Keep YT-DLP meticulously updated to avoid 403 Forbidden errors or slow downloads.

📖 Manual Troubleshooting
If auto-update fails, run directly:
python -m pip install -U yt-dlp

If YouTube gives 'Sign in' warnings:
Use cookies by extracting your browser cookies and modifying `modules/youtube_downloader.py`.
🖥 GPU Hardware Acceleration (CUDA/NVENC)
💡 Setting Up GPU Encoding (NVENC)
Hardware acceleration offloads video rendering to your NVIDIA GPU, cutting merge/watermark times by ~85%.

  1. Install latest NVIDIA Graphic Drivers on your machine.
  2. Ensure the FFmpeg installed natively supports `h264_nvenc` (Our Auto-Installer does!).
  3. Status is automatically verified through the tracker above.
  4. If NVIDIA is detected, app processes natively route heavily through the GPU.
