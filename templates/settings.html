System & GPU
🖥 GPU Hardware Acceleration
💡 Setting Up GPU Encoding (NVENC)
Hardware acceleration offloads video rendering to your NVIDIA GPU, cutting merge/watermark times by ~85%.

  1. Install latest NVIDIA Graphic Drivers on your machine.
  2. Download an FFmpeg build with NVENC support (e.g. from Gyan.dev or BtbN).
  3. Extract the FFmpeg `.exe` files into a folder (like `C:\ffmpeg\bin`).
  4. Add that folder to your Windows System PATH Environment Variable.
  5. Restart Nikethan toolkit using `start server.bat`.
